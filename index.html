📖 Como configurar a atualização automática (Apps Script)

⚖️ Script da Trilha Compliance

  1. Abra a planilha do formulário de Compliance
  2. Clique em Extensões → Apps Script
  3. Apague o código existente e cole o conteúdo do arquivo apps_script_compliance.js
  4. Salve (Ctrl+S)
  5. Clique em Implantar → Nova implantação
  6. Tipo: App Web
  7. Executar como: Eu mesmo
  8. Quem pode acessar: Qualquer pessoa
  9. Clique em Implantar e autorize o acesso
  10. Copie a URL do app Web gerada
  11. Cole no campo "API Compliance" no topo do painel

⚠️ A URL parece algo como: https://script.google.com/macros/s/AKfy.../exec

🛡️ Script da Trilha Guardiões de Dados

  1. Abra a planilha do formulário de Guardiões de Dados
  2. Repita os mesmos passos acima usando o arquivo apps_script_dados.js
  3. Copie a URL gerada e cole no campo "API Guardiões" no painel

💡 Cada planilha precisa do seu próprio script e URL.


🔄 Atualização automática

  1. Com as URLs salvas, o painel consulta as APIs automaticamente a cada 5 minutos
  2. Você também pode clicar em "↺ Atualizar agora" a qualquer momento
  3. As URLs ficam salvas no navegador (localStorage) — não precisa redigitar

💡 Quando um novo participante preenche o formulário, o painel reflete em até 5 min.
